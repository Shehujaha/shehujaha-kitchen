# 🍳 Shehujaha Kitchen - Admin Control Panel

A sleek, responsive, and animated dark-mode administration platform built for managing food inventory, delivery zones, and electronic payment configurations for **Shehujaha Kitchen**, based in Gidan Shehujaha, Rano Local Government, Kano State.

## 🚀 Live Features
* **Food Catalog Inventory:** Dynamic listing controls to handle stock status toggles, pricing structures, and item categorization.
* **Logistical Delivery Scope:** Configurable regional shipping matrices mapped specifically for Kano State areas with automatic rate conversions.
* **Payment API Gateways:** Direct merchant pipeline tracking with built-in toggle hooks for platforms like Paystack and Stripe.
* **Disaster Recovery Suite:** Point-in-time state diagnostic dumps that compile database properties into structured secure `.sql` script freezes.

## 🛠️ Tech Stack Architecture
* **Backend Runtime Engine:** Node.js paired with the Express routing framework.
* **Frontend User Interface:** Alpine.js reactive components, Tailwind CSS styling arrays, and FontAwesome structural vector iconography.

## 🌐 Production Cloud Deploy Setup (Render)
To launch this engine onto live production infrastructure without manual installation states:
1. **Runtime Selection:** Node
2. **Build Routine:** `npm install`
3. **Initialization Routine:** `node server.js`
